Agent Config Setup

Download and run the installer script from this server. The script syncs your config, installs packages with uv, and links skills into Cursor.

Run in terminal 
curl -fsSL https://agent-config.neetbyte.fun/setup-agent-config.sh | bash -s -- --source remote

  1. Copy the command above.
  2. Paste and run it in your shell.
  3. Re-open terminal, then run agent_config_update later for upgrades.
